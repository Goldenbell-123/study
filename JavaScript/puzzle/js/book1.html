• 자바스크립트
      • js는 웹 페이지에 동적 기능을 추가하기 위해 사용된다.

      외부 .js 파일을 연결하려면 <script src=" js 파일 주소 "></script>로 사용할 수 있다.
      src로 불러온 스크립트 내부에는 그 어떤 내용도 적을 수 없다. ( 주로 "./root/*.js" 경로로 활용한다. )

      js는 주로 웹 페이지를 움직이기 위해 사용되지만, 포토샵, 오픈오피스, 구글 앱스, 서버 프로그래밍에서도 어플리케이션 스크립트를 작성하기 위해 사용될 수 있다.

      변수를 선언할 때 시작부에는 반드시 ( 문자, _ , $) 로 시작한다. ( 하지만, 추후 Jquery와의 혼동이 없도록 $는 쓰지 않는다. )
      Camel Case : 낙타의 등 처럼 울록볼록한 모양의 형태 ex) userName
      Underbar Case : 단어마다 _ 를 이어주며 변수명을 짓는 형태 ex) user_name
