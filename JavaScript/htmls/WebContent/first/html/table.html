TABLE

  • table : 테이블을 정의
  • caption : 테이블의 제목표시
  • tr (table row) : 테이블 내에 한 행 정의
  • th (table heading) : 셀 제목으로 지정하여 강조 처리
  • td (table data) : 실제 데이터 표시
