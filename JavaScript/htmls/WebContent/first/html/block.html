HTML BLOCK elements

대부분의 html 요소는 블럭
  • div :
    • 논리적인 섹션으로 구분할 때 사용
    • 자체로는 의미 없음
    • 블럭 수준 요소여서 한 줄 전체 차지
  • span:
    • 인라인요소
    • 한 줄 전체를 차지하지 않음
    • 텍스트를 위한 컨테이너로 사용되거나
      텍스트 일부의 스타일 속성 설정에 사용
