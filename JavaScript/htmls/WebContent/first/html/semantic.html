html5의 의미 기반 태그들

태그이름 설명
header
  • 머리말 부분 지정
  • 사이트 전체의 머리말 사용
  • header 태그 내에서 hn 태그를 사용
hgroup
  • 섹션의 제목
  • 섹션의 제목이 여러 단계일 때 사용
nav
  • 네비게이션을 위해 구성
  • 주요 링크들 그룹화
article
  • 문서의 실제 내용
section
  • 컨텐츠 그룹화
aside
  • 본문을 표시하고 남는 내용들
  • 관련 사이트 링크나 광고 등
footer
  • 주로 페이지 하단에 표시
  • 제작자 정보 및 저작권 정보 등
figure
  • 그래픽이나 이미지를 위한 컨셉 표시
