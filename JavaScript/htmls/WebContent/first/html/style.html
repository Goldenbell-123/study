 	전체 선택자 	 * 		페이지 내의 모든 요소에 적용
 	타입 선택자 	태그명  	적용할 HTML 태그에 
 	아이디 선택자 	#아이디 	HTML 태그에 부여된 id 속성
 					선택시 사용
 					하나의 요소만 선택 가능
 	클래스 선택자	.클래스		HTML 태그에 부여된 class 속성
 					여러 개의 요소 선택 가능
 	
 										
 	그룹 선택자		,	로 구분하여 선택자를 나열
 	자식 선택자		>	로 구분하여 선택자를 나열
 	후손 선택자 		공백	로 구분하여 선택자를 나열
 	형제 선택자		~	로 구분하여 선택자를 나열	
 	
 	
 	속성 선택자     태그이름[]	특정 속성을 갖는 요소 선택
